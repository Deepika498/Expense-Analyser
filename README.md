# Expense Analyzer 💸

A web-based Expense Analyzer application built using Flask that helps users track, manage, and analyze their daily expenses efficiently.

## 🚀 Features

* Add daily transactions
* View expense reports
* User authentication (Login / Register)
* Organized UI using templates
* Store data using SQLite database


## 🛠️ Tech Stack

* Backend: Python (Flask)
* Frontend: HTML, CSS
* Database: SQLite

## 💻 How to Run

1. Clone the repository: git clone https://github.com/Deepika498/Expense-Analyser.git


2. Navigate to the project folder: cd Expense-Analyser


3. Install dependencies: pip install flask


4. Run the application: python app.py


5. Open in browser: http://127.0.0.1:5000
